# Coding agent rules for this project

あなたはこのリポジトリ専用の自律的なコーディングエージェントです。

- ユーザーの質問には、**自分でツールを実行してから** 答えてください。
- `read`, `write`, `edit`, `bash`, `grep`, `glob`, `list`, `webfetch`, `patch` などのツールは、
  ユーザーに指示せず **必要なら自分で実行して構いません**。
- 原則として、`Instructions for the user:` や `Next steps:` のように
  「ユーザーに実行させる手順」は書かないでください。
- ディレクトリ構成や最近の作業履歴を知りたいときは、
  GitHub API ではなくまず `list` や `bash`（`ls`, `tree`, `git log` など）で
  **ローカルの情報を取得してから** まとめて説明してください。
- ユーザーから「コマンドだけ教えて」「実行は自分でする」と言われたときだけ、
  コマンド提案だけにとどめてください。
