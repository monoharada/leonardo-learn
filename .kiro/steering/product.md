# Product Overview

デザインシステム向けのカラーパレット生成ツール。OKLCH色空間を使用し、WCAGアクセシビリティ基準を満たしながら知覚的に均一な色パレットを生成する。

## Core Capabilities

1. **コントラスト比駆動の色生成**: 目標コントラスト比を起点として色を生成する逆引きアプローチ（Adobe Leonardo方式を継承）
2. **OKLCH色空間サポート**: 知覚的に均一な色空間により、従来のLCH/LAB/HSVより自然な色遷移を実現
3. **WCAGコンプライアンス**: WCAG 2.1/2.2（標準的なコントラスト比）およびWCAG 3 APCA（知覚的コントラスト）に対応
4. **動的パレット調整**: lightness/contrast/saturationパラメータによるリアルタイム適応
5. **日本語ネイティブUI**: UIとドキュメントを日本語で提供（英語対応も視野）

## Target Use Cases

- **デザインシステム構築**: 企業・プロダクトのカラーパレット設計
- **アクセシビリティ検証**: WCAGコントラスト要件を満たす色の自動生成
- **テーマ切替**: ライトモード/ダークモードの一貫した色スケール提供
- **プロトタイピング**: 迅速なカラースキーム探索

## Value Proposition

**アクセシビリティとデザイン品質の両立**: 従来の「色を選んでコントラストを確認」ではなく、「コントラスト要件から色を生成」することで、視覚的に美しく、かつWCAG準拠を保証するパレットを効率的に作成できる。

**モダン色空間による知覚的均一性**: OKLCHの採用により、従来のRGB/HSLでは実現困難だった、人間の視覚に即した自然な色遷移と広色域対応を実現。

**日本語環境での最適化**: 日本語フォント・UI環境でのコントラスト知覚を考慮した設計。
