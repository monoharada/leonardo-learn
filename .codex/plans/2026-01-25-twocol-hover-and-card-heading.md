# 行政サービス案内：右リンクカードのホバー統一＋左カード見出し追加

## 目標
- 右カラムのカード型リンク（手続きカテゴリ）に、左カード（イラストカード）と同系統のホバー反応（枠色＋アウトライン風）を付ける
- 左カード上にも見出しを追加して、右の「手続きカテゴリ」と縦位置・余白のアラインを揃える
- DADSの `:focus-visible` を壊さない（見切れ/過剰装飾なし）

## 背景
- 左カードは hover / focus-within で `border-color` と `box-shadow: 0 0 0 2px` により「押せる」反応が明確。
- 右のリンクリストはカード化したが、ホバー反応が左カードと揃っていない。
- 2カラム内で右だけ見出しがあり、左カード側の先頭位置が揃って見えない。

## スコープ
- やること：
  - `src/ui/demo/views/palette-preview.ts` に左カード用の見出し（h3）を追加
  - `src/ui/styles/components.css` で右リスト項目の hover を左カードと同様の反応に揃える
  - 必要なら `.preview-twocol__right` を `flex + gap` にして余白スケールを統一
- やらないこと：
  - 選択/現在地の強調の追加
  - 配色生成/色計算ロジックの変更

## 前提 / 制約
- hover は `@media (hover: hover)` 内で適用（タッチ環境で暴れない）
- 色は `--preview-link` / `--preview-border` の範囲で（ハードコード色を増やさない）
- `:focus-visible` は DADS既定を尊重（必要以上に重ねない）
